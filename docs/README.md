# DOCUMENTS

PixelRadio is a DiY FM Radio Transmitter with RDS (Radio Data System) capabilities. It was developed for holiday "Pixel" displays (e.g., animated Christmas lights).

This folder contains all the project's documents.

## ARTWORK FOLDER

Printer Circuit Board Files for Commercial PCB Fabrication are stored in the [Artwork folder](./Artwork).

## HARDWARE FOLDER

Project assembly information is stored in the [Hardware folder](./Hardware).

Contents Include:\
``Parts List (Bill-of-Materials)`` \
``Supplier/Vendor List`` \
``Assembly Tips``

## IMAGES FOLDER

All images used in the documentation are stored in the [Images folder](./Images).


## USER MANUAL FOLDER

The Instruction Manual Documents are stored in the [User Manual folder](./User_Manual).
